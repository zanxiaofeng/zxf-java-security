# pem format for public key, private key and cert
-----BEGIN PUBLIC KEY-----
...Base64 encoded key...
-----END PUBLIC KEY-----

-----BEGIN RSA PUBLIC KEY-----
...Base64 encoded key...
-----END RSA PUBLIC KEY-----

-----BEGIN PRIVATE KEY-----
...Base64 encoded key...
-----END PRIVATE KEY-----

-----BEGIN RSA PRIVATE KEY-----
...Base64 encoded key...
-----END RSA PRIVATE KEY-----

-----BEGIN CERTIFICATE-----
...Base64 encoded key...
-----END CERTIFICATE-----

-----BEGIN CERTIFICATE REQUEST-----
...Base64 encoded key...
-----END CERTIFICATE REQUEST-----
